## 📌 Features

- Create and manage tasks
- Task statuses:
  - Created
  - In progress
  - In review
  - Completed
  - Rejected
- Priority levels:
  - Low
  - Medium
  - High
  - Critical
- Task modification:
  - Change status
  - Change priority
  - Edit task data
- Task retrieval by ID
- Task removal
- Undo / Redo support via History Manager
